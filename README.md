# Mimalodi Visual Data Space

A Django-based web application for blogging and data visualization.

## Features
- User authentication
- Blog system
- Categories
- Admin dashboard
- Data visualization tools (coming soon)

## Tech Stack
- Python
- Django
- HTML, CSS, Bootstrap
- SQLite

## Setup Instructions

```bash
git clone https://github.com/Raversclavers/MimalodiVisualDataSpace.git
cd MimalodiVisualDataSpace
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
